# Event Management System

A full-stack web application developed using Angular, Spring Boot, and PostgreSQL for efficient event planning and management.

## 🚀 Features
- User registration and login system  
- Create, update, and delete events  
- Event booking and management  
- Responsive and user-friendly interface  
- Secure backend with REST APIs  

## 🛠️ Technologies Used
- Frontend: Angular  
- Backend: Spring Boot  
- Database: PostgreSQL  
- API: RESTful Web Services  

## 📁 Project Structure
- Event management UI (Frontend - Angular)  
- Event management API (Backend - Spring Boot)
